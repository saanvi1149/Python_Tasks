# Python Basics Programs

This repository contains a collection of 10 basic Python programs. The programs cover topics such as variables, operators, conditional statements, loops, functions, lists, dictionaries, and string manipulation.

### Programs Included

* Information using Variables
* Arithmetic Operations on Two Numbers
* Even or Odd Number Checker
* Largest of Three Numbers
* Numbers from 1 to 100 using Loops
* Factorial Calculator
* List Operations
* Vowel Counter
* Student Dictionary
* Palindrome Checker
